# Contributing to Seovela

We welcome contributions! Here's how to get started:

## Development Setup
1. Clone the repo into `wp-content/plugins/`
2. Run `composer install` (if applicable)
3. Run `npm install` (if applicable)
4. Activate the plugin in WordPress

## Pull Requests
- Fork the repo and create a feature branch
- Follow WordPress coding standards (PHPCS with WPCS)
- Test your changes on a fresh WordPress install
- Submit a PR with a clear description

## Reporting Bugs
- Use GitHub Issues
- Include WordPress version, PHP version, and steps to reproduce
