## ⚙️ Global Meta Settings

### 1. **Robots Meta**
Configure default robots directives for your entire site:

- **Index** - Allow search engines to index pages (default)
- **No Index** - Prevent indexing (useful for staging sites)
- **No Follow** - Prevent search engines from following links
- **No Archive** - Hide cached links in search results
- **No Image Index** - Prevent image indexing
- **No Snippet** - Disable snippets in search results

### 2. **Advanced Robots Meta**
Fine-tune how search engines display your content:

- **Max Snippet** (-1 to ∞)
  - Control snippet character length
  - -1 = no limit

- **Max Video Preview** (-1 to ∞)
  - Set max video preview duration in seconds
  - -1 = no limit

- **Max Image Preview** (None/Standard/Large)
  - Control image preview size
  - Large = best for visual content

### 3. **Archive Settings**

- **Noindex Empty Archives**
  - ✅ Automatically noindex empty category/tag pages
  - Prevents thin content issues
  - Auto-updates to index when posts are added

### 4. **Title Settings**

- **Separator Character**
  - Choose from: - | ~ • > <
  - Use `%separator%` or `%sep%` in titles
  - Example: `Post Title %sep% Site Name`

- **Capitalize Titles**
  - ✅ Auto-capitalize first character of each word
  - Makes titles more professional

## 💡 Pro Tips

### For Best SEO:
1. ✅ Keep default "Index" for most sites
2. ✅ Enable "Noindex Empty Archives"
3. ✅ Set Max Snippet to -1 (no limit)
4. ✅ Use Large Image Preview
5. ✅ Upload high-quality OpenGraph image

### For Social Sharing:
1. ✅ Upload 1200x630px fallback image
2. ✅ Use "Summary Card with Large Image"
3. ✅ Test with Facebook Debugger
4. ✅ Test with Twitter Card Validator

### For Titles:
1. ✅ Choose a clean separator (- or |)
2. ✅ Use %sep% in title templates
3. ✅ Consider enabling capitalize
4. ✅ Keep titles under 60 characters
